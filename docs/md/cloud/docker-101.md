## docker cli

```bash
$ docker image ls
$ docker images
$ docker image prune
$ docker rmi username/frontend:0.0.1
$ docker rmi f8acfb3d1737

$ docker ps
$ docker ps -a --no-trunc

$ docker start gracious_lovelace
$ docker inspect gracious_lovelace
$ docker stop gracious_lovelace
$ docker rm -v gracious_lovelace

$ docker volume ls
$ docker volume prune
```
